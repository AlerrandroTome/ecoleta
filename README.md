# Ecoleta
Ecoleta is a platform that allows you to register and search for waste collection points.

## Stack used
**Front-end:** Typescript and ReactJS.
**Back-end:** Typescript, NodeJS and ExpressJS.
**Mobile:** Typescript and React Native.
